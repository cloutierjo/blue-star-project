HOW TO :
    * Partir le contrôleur serveur. Le serveur sera démarrée
    * Partir le contrôleur client L'interface sera démarrée
    * De la il faut ouvrir un projet et naviguer dans le projet par les onglets
    *****N.B. IMPORTANT****
    	VOUS DEVEZ SAUVEGARDER LE PROJET AVANT DE QUITTER OU DE FERMER LE PROJET
    
    
    
    ( Les composantes les plus complexes seront expliqué ici)
    
    POUR PLANNING:
    	* Pour Afficher les détails d'un Planning General : cliquer sur bouton "Afficher Details" associé
    	* Les CRC sont afficher dans le coin inférieur droit
    	
    POUR SCRUM:
    	* Pour afficher le planning détaillé du sprint associé à la date entrée dans le comboBox du SCRUM:
    		1) Entrez  une date dans le format demandé (yyyy-mm-dd) 
    		2) appuyer sur la touche "enter"
    	
    	* Pour creer un nouveau SCRUM:
    		1) Entrez la date du nouveau scrum
    		2) appuyez sur enter
    		3) choississez un utilisateur
    		4) remplir les champs
    		5) si nécessaire changer d'utilisateur et remplissez de nouveau les champs


CONFIGURATION MINIMUM : 
	* la configuration minimale de Python/TKinter
	* résolution : 1024x768
	* connexion réseau si utilisé avec le composant réseau
	
BUGS CONNUS :
    
-	!!! ATTENTION !!!

    L'aplication ne fonctionne pas correctement avec les versions de python 
    précédant python 2.6.2, l'aplication peut potentiellement être fonctionnelle
    sur python 2.5.x et sur python 2.6.1, mais n'a pas été entièrement testé parce que
    ces version causaient un problème au niveau des couleurs dans Tix. 
    
    Si vous avez une version incompatible (Donc 2.6.1 en descendant) vous devez 
    désinstaller votre version actuelle puis installer python 2.6.2 ou plus (par 
    contre l'application n'est pas compatible avec python 3.x), faire seulement
    une mise à jour ne corrige pas les problèmes.
        
    En gros :
    * Python 2.5.x semble fonctionnel, mais non supporté par les developpeurs
    * Python 2.6.1 : Ne marche pas
    * Python 2.6.2 : WORKS
    * Python 3.x : Ne marche pas car certaines syntaxes ont changées

    python 2.6.2 : http://www.python.org/download/releases/2.6.2/
    
-	L'affichage n'est pas correcte (et non supporté pour le moment) sur les platformes autre que windows, certaines 
	informations sont manquantes dans les vue d'analyses
